# My Computer Configuration

## About
This repository serves as a centralized collection of personal computer configuration files, automation scripts, and system optimization tools. It aims to streamline the setup of new machines, maintain consistent environments across different systems, and automate routine tasks for enhanced productivity.

## What's Included
- **Dotfiles**: Configuration files for various applications (e.g., Bash, Zsh, Vim, Git).
- **Installation Scripts**: Automated scripts for installing software packages and setting up development environments.
- **System Utilities**: Small scripts and tools for system maintenance, monitoring, and optimization.
- **Backup & Restore**: Scripts to manage backups of important configurations and data.

## How to Use

### Cloning the Repository
```bash
git clone https://github.com/Prem2868/my-computer-config.git
cd my-computer-config
```

### Applying Configurations
Refer to the specific directories and their respective `README.md` files for detailed instructions on how to apply individual configurations or run automation scripts. For example:

```bash
# To set up dotfiles
./scripts/setup_dotfiles.sh

# To install common development tools
./scripts/install_dev_tools.sh
```

**Warning**: Always review scripts and configuration files before execution to understand their impact on your system.
